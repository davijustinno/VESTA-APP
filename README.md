# 📚 Sobre o Projeto

O **VESTA** é uma plataforma de estudos gamificada inspirada na experiência do **Duolingo**, desenvolvida para preparação para:

- 🇧🇷 ENEM
- 🎓 PAS
- 📖 Fuvest
- 🏛️ Unicamp
- 📚 UEM
- 🎯 UFPR
- e diversos outros vestibulares.

O objetivo é transformar o estudo em uma experiência divertida, personalizada e extremamente eficiente.
